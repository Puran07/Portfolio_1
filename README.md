# Link Page

Static Linktree-style landing page for Puran Singh Rathore.

## Files

- `index.html`: page structure
- `styles.css`: visual styling
- `site.js`: all editable contact data and link rendering
- `puran-singh.vcf`: downloadable contact card used by the Save Contact button
- `profile-photo.svg`: placeholder image used when no real photo is present

## Update Your Details

Edit the `profile` object at the top of `site.js`:

- phone number
- WhatsApp number
- email
- website
- Instagram
- LinkedIn
- Google Maps link
- location label

## Replace The Photo

Add your real image as `profile-photo.jpg` in the project root. The page will use that automatically and fall back to `profile-photo.svg` if the JPG is missing.

## Free Hosting

### GitHub Pages

1. Create a new GitHub repository.
2. Upload these files.
3. Open repository `Settings`.
4. Go to `Pages`.
5. Deploy from the `main` branch.

### Netlify

1. Create a free Netlify account.
2. Drag this folder into Netlify.
3. Your page will be live instantly.
