# Link Page

Static Linktree-style landing page for Arjun Singh.

## Files

- `index.html`: page structure
- `styles.css`: visual styling
- `site.js`: all editable contact data and link rendering
- `profile-photo.svg`: placeholder image you can replace with a real profile photo

## Update Your Details

Edit the `profile` object at the top of `site.js`:

- phone number
- WhatsApp number
- email
- website
- Instagram
- LinkedIn
- Google Maps link
- location label

## Replace The Photo

Swap `profile-photo.svg` with your real image and keep the same filename, or update the image path in `index.html`.

## Free Hosting

### GitHub Pages

1. Create a new GitHub repository.
2. Upload these files.
3. Open repository `Settings`.
4. Go to `Pages`.
5. Deploy from the `main` branch.

### Netlify

1. Create a free Netlify account.
2. Drag this folder into Netlify.
3. Your page will be live instantly.
